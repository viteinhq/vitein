# Contributing to vite.in

Thanks for your interest in contributing. vite.in is an open-core project (AGPL-3.0) and we welcome issues, discussions, and pull requests.

Before you start:

- Read the [Code of Conduct](./CODE_OF_CONDUCT.md).
- Browse existing [issues](https://github.com/viteinhq/vitein/issues) and [discussions](https://github.com/viteinhq/vitein/discussions) before opening a new one.
- For non-trivial changes, open a discussion or issue first so we can align on scope.
- By submitting a PR, you agree to sign the CLA (EasyCLA bot will prompt you).

Architecture and roadmap context lives in [`docs/`](./docs/) (ARCHITECTURE.md, ROADMAP.md, PROJECT_PLAN.md).

Security issues: please do **not** file a public issue. See [SECURITY.md](./SECURITY.md).
